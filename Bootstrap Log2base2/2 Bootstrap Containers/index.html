Fixed-Width Container

To create a fixed width layout, we can use the "container" class. This container is created using the "container" class and is horizontally centered with a specific width.

Full-width container

To span our content to the full widht of the screen, we can use the "container-fluid" class. This container is creatged using the "container-fluid" class and takes up the entire width of the viewport, utilizing the available screen widht.
